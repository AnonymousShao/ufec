# ufec
ustack front end component

## Install

```bash
npm i ufec --save
```

## Api

### main

Main layout.

### modalV2

Main modal component.

### modalPlus

Main modal plus.

### modalDelete

Main modal delete.

### modalAlert

Main modal alert.

### sideMenu

Side menu component.

### subs `{Object}`

A list of subs components.

```javascript
import {subs} from 'ufec';
const Steps = subs.Steps;
```

### history

React-router's history state.
